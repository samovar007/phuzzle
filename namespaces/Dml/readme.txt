dml - debug mini library - сборник полезных функций для отладки и не только


+ получение переменной из окружения, например для установки режима отладки
+ логер отладки с выводом на экран (консоль или html)
+ работа со строками (бектрейс и пр.)
+ посмотреть, что еще умеет error_log
- ловля ошибок и работа с перехватом исключений
- ловля сигналов
+ посмотреть на автозагрузчик
